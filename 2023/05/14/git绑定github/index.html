git绑定github

发表于 2023-05-14 分类于 git

设置全局账户名和邮箱

  • 使用 git config --global user. xxx 命令进行设置，内容可以随意填写
  • 使用 git config -l 检查设置情况

生成 ssh key

  • 使用 ssh-keygen -t rsa ,连续敲击回车键
  • 进入 .ssh 文件夹，找到 id_rsa.pub ，复制其中的内容

绑定github

  • 进入github的 Settings 中，点击 SSH and GPG keys ，再点击左上角绿色的 New SSH key ，在 key 中粘贴 id_rsa.pub 中的内容，最后点击绿色的 Add SSH key
