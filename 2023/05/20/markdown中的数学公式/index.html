markdown中的数学公式

发表于 2023-05-20 更新于 2023-05-23 分类于 学习记录

1. 插入公式的方法

1.1 文中插入公式

  • $数学公式$
    e.g. $y=ax+b$
    效果：$y=ax+b$

1.2 独立插入公式

  • $$数学公式$$
    e.g. $$y=ax+b$$
    效果：$$y=ax+b$$

2. 常见的数学符号

2.1 上下标

  • 上标：$底数^{指数}$
    e.g. $x^{n}$
    效果：$x^{n}$
  • 下标：$底数_{指数}$
    e.g. $x_{1}$
    效果：$x_{1}$

2.2 括号与分隔符

名称 语法 效果 名称 语法 效果
左尖括号 $\langle$ $\langle$ 右尖括号 $rangle$ $\rangle$
左花括号 $\{$ ${$ 右花括号 $\}$ $}$
左中括号 $[$ $[$ 右中括号 $]$ $]$
左括号 $($ $($ 右括号 $)$ $)$

2.3 分数

  • $\frac{分子}{分母}$
    e.g. $\frac{y}{x}$
    效果：$\frac{y}{x}$
  • 分数较复杂时，采用 $分子 \over 分母$
    e.g. $a+b+c+d \over e+f+g+h$
    效果：$a+b+c+d \over e+f+g+h$
  • 只有两个字符时，也可采用 $\frac ab$
    效果：$\frac ab$

2.4 开方

  • $\sqrt[根指数，默认是2，可不写]{被开方数}$
    e.g. $\sqrt{x}$
    效果：$\sqrt{x}$
    e.g. $\sqrt[4]{x}$
    效果：$\sqrt[4]{x}$

2.6 积分

  • $\int_{积分下限}^{积分上限} {被积表达式}$
    e.g. $\int_{1}^{2} {\frac{1}{x}}$
    效果：$\int_{1}^{2} {\frac{1}{x}}$

2.7 无穷

名称 语法 效果
无穷 $\infty$ $\infty$
正无穷 $+\infty$ $+\infty$
负无穷 $-\infty$ $-\infty$

2.8 极限

  • $$\lim_{变量 \to 表达式} 表达式$$
    e.g. $$\lim_{n \to +\infty} \frac {1}{n}$$

2.9 累加和累乘

  • 累加：$\sum_{下标表达式}^{上标表达式} {表达式}$
    e.g.$$\sum_{n=0}^{+\infty} {x^{n}}$$
    效果：$$\sum_{n=0}^{+\infty} {x^{n}}$$
  • 累乘：$\prod_{下标表达式}^{上标表达式} {表达式}$
    e.g. $$\prod_{n=1}^{+\infty} {n}$$
    效果：$$\prod_{n=1}^{+\infty} {n}$$

2.17 为公式添加注释

  • \test{注释}
    e.g. $f(x)=\begin{cases} 0, & \text{x > 0} \\ 1, & \text{x < 0} \end{cases}$
    效果：
    $f(x)=\begin{cases} 0, & \text{x > 0} \ 1, & \text{x < 0} \end{cases}$
